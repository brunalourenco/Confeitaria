# Sistema para Confeitaria 
A Confeitare é uma pequena confeitaria artesanal (fictícia) criada para transformar momentos simples em experiências doces e inesquecíveis. Nossa essência nasce do cuidado em cada detalhe: ingredientes selecionados, técnicas que unem tradição e modernidade e um toque especial que só o verdadeiro amor pela confeitaria pode oferecer.

Trabalhamos com bolos personalizados, doces finos, sobremesas exclusivas e produtos sob encomenda, sempre prezando pela qualidade, frescor e apresentação impecável. Aqui, cada criação é feita como se fosse para alguém especial — porque para nós, você é.

## Objetivos
### Objetivos Geral
Desenvolver um sistema front-end para a Confeitare que apresente a marca, seus produtos e serviços de forma moderna, simples e intuitiva utilizando apenas HTML e CSS.

### Objetivos específicos
* Criar uma landing page atrativa para divulgação de produtos;
* Apresentar os produtos em layout organizado e visualmente agradável;
* Facilitar o contato com o cliente;
* Apresentar formas de encomendar produtos; 

## Justificativa
O sistema da Confeitare foi idealizado para fins acadêmicos, com o objetivo de aplicar na prática conceitos de desenvolvimento front-end utilizando HTML e CSS. Além disso, o projeto permite explorar identidade visual, organização de layout, tipografia, cores e estruturação de páginas web, simulando um cenário real de negócio.

A escolha por uma confeitaria se justifica pela forte relação entre estética, experiência do usuário e comunicação visual, fatores essenciais no desenvolvimento de interfaces modernas.

## História de usuário

* Como cliente, quero visualizar os produtos com fotos e descrições para escolher o que desejo comprar.

* Como cliente, quero acessar facilmente as informações de contato para realizar encomendas.


* Como visitante, quero conhecer a história da Confeitare para criar conexão com a marca.

* Como cliente, quero ver os valores e diferenciais da confeitaria para confiar no serviço.

# Restrições do projeto
O sistema deverá ser desenvolvido apenas com HTML e CSS;

Não será permitido o uso de JavaScript;

Não será utilizado banco de dados;

O projeto terá finalidade acadêmica;

O conteúdo será fictício.

# Considerações Finais

O projeto Confeitare representa a união entre design, organização e desenvolvimento front-end, servindo como base para estudos e aprimoramento de habilidades em criação de interfaces web.
